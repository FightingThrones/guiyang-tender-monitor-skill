# Roadmap

## v0.2.0 - Nationwide SME Opportunity Scoring

- Improve SME-friendly scoring rules.
- Add budget range classification.
- Add buyer type classification.
- Add province/city extraction.
- Add deadline countdown.

## v0.3.0 - Structured Tender Parsing

- Extract qualification requirements.
- Extract scoring method and price/technical/commercial weights.
- Extract bid opening and file acquisition windows.
- Separate official announcements from aggregator leads.

## v0.4.0 - Lead-To-Draft Workflow

- Connect monitor output to Tender Bid Assistant.
- Generate tender summaries suitable for bid drafting.
- Export selected opportunities as input files for draft proposal generation.

## v0.5.0 - Dashboard

- Add a local HTML dashboard.
- Add filters by province, buyer type, procurement mode, budget, and deadline.
- Add project status tracking.

## v1.0.0 - National Tender Agent

- Stable nationwide monitoring workflow.
- Configurable region and industry profiles.
- Daily email/report automation.
- Human review workflow and opportunity follow-up notes.
